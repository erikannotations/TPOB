This repository includes the full metadata of the Thomas Pynchon Bibliography (TPOB) available on Zotero:

https://www.zotero.org/groups/5470223/thomas_pynchon_online_bibliography/library

To add Pynchon-related articles, books, etc., or to suggest an edit/correction to an existing entry, please use the following form: https://forms.office.com/e/KiX0hQgeWN


**Files:**

Full export of the 1.0 release in formats .csv, BibTeX, and BibLaTeX
