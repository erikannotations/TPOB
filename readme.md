This repository includes the full metadata of the Thomas Pynchon Bibliography (TPOB), a bibliometric resource for Pynchon studies.

TPOB is introduced in the journal article,

* Erik Ketzan, “A Bibliometric Overview of Pynchon Studies: Introducing the Thomas Pynchon Online Bibliography (TPOB)”. *Orbit: A Journal of American Literature* 12:1 (2025), https://doi.org/10.16995/orbit.23348
* https://orbit.openlibhums.org/article/id/23348/

TPOB is available on Zotero:

* https://www.zotero.org/groups/5470223/thomas_pynchon_online_bibliography/library

To add Pynchon-related articles, books, etc., or to suggest an edit/correction to an existing entry, please use the following form: https://forms.office.com/e/KiX0hQgeWN
